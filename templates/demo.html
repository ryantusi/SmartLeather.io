How to use our software?

Read the STEPS and scroll

Step 1: View Live Dashboard

  1. Access the Dashboard:
    Log in to the software platform using your credentials. Navigate to the dashboard section to gain an overview of real-time analytics.
  2. Analyze Data:
    Utilize interactive tools on the dashboard to analyze key metrics. Explore different visualizations and filters to gain insights into specific aspects of the data.
  3. Download Raw Data:
    Download raw data for further analysis by selecting the download option. Choose specific data sets or export the entire dataset for more in-depth examination.

Step 2: Fill in Forms

  1. Form Submission:
    Locate and access the respective forms for product, customer, or order management. Fill in the required information accurately, ensuring completeness of the form.
  2. Unique ID Generation:
    Upon successful creation, each product, customer, or order will be assigned a unique identification number. This ID serves as a reference for future interactions and records.
  3. QR Code for Orders:
    Specifically for order creation, the system will generate a QR code for each order. Save the QR code to facilitate easy tracking and for deletion purposes if required.

Step 3: Start Your Scanning Demo

  1. Order Creation:
    Initiate the scanning demo by creating a new order through the designated interface. The system will generate a unique QR code for the newly created order.
  2. Live Dashboard Updates:
    Observe the live dashboard as it dynamically updates in response to the newly created order. Changes in key metrics, visualizations, and relevant data points will be reflected in real-time.
  3. Data Monitoring:
    Track the journey of the order through the live dashboard, noting any shifts or updates. Use this hands-on experience to understand the software's responsiveness and real-time data monitoring capabilities.
